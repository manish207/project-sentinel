# Roadmap

Status of Milestones for Project Sentinel:

- [x] **Milestone 1 — Core Domain**: Entities, CRUD, Validation
- [x] **Milestone 2 — Hierarchy**: Parent/Child Tasks, Task Trees
- [x] **Milestone 3 — Recurring Tasks**: Daily, Weekly, Monthly, Yearly
- [x] **Milestone 4 — Dependency Graph**: Dependencies, Ready Tasks, Cycle Detection
- [ ] **Milestone 5 — Intelligent Scheduler** (In Progress): Recommends what to work on next
- [ ] **Milestone 6 — Calendar Engine**: Today/Tomorrow/Week/Month views, automatic time blocking
- [ ] **Milestone 7 — Kanban Board**: Inbox, Planned, Ready, In Progress, Waiting, Completed columns
- [ ] **Milestone 8 — Eisenhower Matrix**: Classify tasks into Do/Schedule/Delegate/Eliminate
- [ ] **Milestone 9 — Critical Path Analysis**: Longest dependency chain, bottlenecks
- [ ] **Milestone 10 — AI Planning Engine**: Context-aware daily planning, workload balancing
- [ ] **Milestone 11 — Notifications & Automation**: Email, GitHub, Google Tasks, TickTick
- [ ] **Milestone 12 — Web Dashboard**: FastAPI backend, React/Tailwind frontend
- [ ] **Milestone 13 — Desktop & Mobile**: Electron desktop, Android, iOS apps
